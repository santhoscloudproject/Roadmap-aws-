# AWS EC2 Static Website Deployment

## Project URL

https://roadmap.sh/projects/ec2-instance

## Description

This project demonstrates deploying a static website on an AWS EC2 Ubuntu instance using Nginx.

## Live Demo

http://65.1.147.81/

## Repository

https://github.com/santhoscloudproject/Roadmap-aws-

## How to Run

1. Launch an Ubuntu EC2 instance.
2. Connect to the instance using SSH.
3. Install Nginx:
   ```bash
   sudo apt update
   sudo apt install nginx -y
   ```
4. Copy the `index.html` file to `/var/www/html/`.
5. Open `http://65.1.147.81/` in your browser.
